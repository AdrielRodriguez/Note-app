## Main Features

- Create, edit, and delete notes.

- Modern, fast, and responsive interface.

- REST API developed with Node.js + Express.

- MongoDB database for persisting notes.

- Frontend/backend communication via Axios/Fetch.

- Structured and easy-to-maintain code.
